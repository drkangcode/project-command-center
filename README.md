# ⚡ Personal Command Center (个人项目控制台)

> 一个基于 Python Streamlit 构建的轻量级、可视化个人项目管理系统。  
> 专为科研人员、开发者及高效能人士打造，集成了任务管理、甘特图、PDCA 闭环及番茄钟功能。

---

## 📖 项目简介

这是一个**完全本地化**的数据隐私优先的项目管理工具。与 Notion 或 Trello 不同，你的所有数据都存储在本地的 CSV 文件中，不会上传到云端，确保了绝对的安全与隐私。

本项目旨在解决多项目并行时的混乱问题，通过可视化的仪表盘、自动化的进度追踪和结构化的日志记录，帮助你掌控生活与工作的每一个细节。

---

## ✨ 核心功能

### 1. 🚀 全局控制中心 (Dashboard)
* **四象限看板**：基于重要/紧急矩阵（Eisenhower Matrix）自动分类任务。
* **KPI 概览**：实时显示总任务数、进行中任务、高优任务及平均进度。
* **燃起图 (Burn-up Chart)**：自动基于日志生成项目进度趋势曲线。

### 2. 📅 项目全景视图
* **交互式甘特图**：直观展示任务的时间跨度与排期。
* **智能 ID 系统**：自动生成项目编号 (如 `STUDY-01`, `WORK-02`) 和子任务编号。
* **子任务权重管理**：支持对子任务进行权重分配 (如 30% + 70%)，精确计算主任务进度。

### 3. 📝 PDCA 每日闭环
* **每日更新**：级联选择“项目 -> 子任务”，记录今日工作内容与贡献进度。
* **进度自动累加**：每日贡献值会自动汇总到主项目的总进度中。
* **周报生成器**：一键抓取本周所有日志，自动生成 Markdown 格式周报，支持下载。

### 4. 🛠️ 效率工具箱
* **番茄钟**：内置可视化倒计时工具，保持专注。
* **全局搜索**：支持搜索任务名、ID、类别及子任务内容，一键跳转详情页。
* **AI 任务拆解**：(演示版) 输入大目标，模拟 AI 辅助拆解子任务流程。

## 💻 如何开始 (Quick Start)

由于本项目采用了**数据隔离**设计（不含 CSV 数据文件），你需要将代码克隆到本地运行。程序会在首次运行时自动为你生成空的数据库。

### 1. 环境准备
确保你的电脑已安装 Python 3.8+。

### 2. 克隆项目
```bash
git clone [https://github.com/你的GitHub用户名/project-command-center.git](https://github.com/你的GitHub用户名/project-command-center.git)
cd project-command-center
```

### 3. 安装依赖
建议创建虚拟环境（Optional but recommended），然后安装依赖库：
```bash
pip install -r requirements.txt
```

### 4. 启动控制台
```bash
streamlit run app.py
```
启动后，浏览器会自动打开 `http://localhost:8501`，你就可以开始使用了！

## 🔒 数据隐私说明

* **完全本地化**：所有数据存储在项目根目录下的 `life_data.csv` (任务数据) 和 `project_logs.csv` (日志数据) 中。
* **Git 忽略**：这两个文件已被配置在 `.gitignore` 中，**绝对不会**被推送到 GitHub。
* **迁移数据**：如果你换了电脑，只需将这两个 CSV 文件复制到新电脑的同名目录下即可。
